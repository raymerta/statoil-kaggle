# STATOIL KAGGLE PROJECT

Project data source: 

https://www.kaggle.com/c/statoil-iceberg-classifier-challenge

How to run the application: 

- Make sure flask is installed (pip install flask). For more information read this: http://flask.pocoo.org/
- store json data both test.json and train.json to the folder
- Run run.sh (./run.sh)
- Open localhost:5000 

## Jupyter notebook

- there's data-exploration notebook file
- it has images from the ship and icebergs also general information about the data

## Playing around with the data

- go to test folder
- test.py generates images from train data
- generator.py gets one data from train data / test data 
- you need to edit its content before run it
