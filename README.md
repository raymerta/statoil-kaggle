# STATOIL KAGGLE PROJECT

Project data source: 

https://www.kaggle.com/c/statoil-iceberg-classifier-challenge

How to run the application: 

- Make sure flask is installed (pip install flask). For more information read this: http://flask.pocoo.org/
- store json data both test.json and train.json to the folder
- Run run.sh
- Open localhost:5000
